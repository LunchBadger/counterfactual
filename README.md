# Counterfactual Account Demo

> Example demos of counterfactual accounts

## Instructions

Lerna symlinks

```bash
make lerna
```

Start ganache server

```bash
make start/testrpc
```

Test contracts

```bash
make test/contracts
```

Deploy contracts

```bash
make deploy/contracts
```

Start hub

```bash
make start/hub
```

Build loopback client for frontend

```bash
make build/client/loopback
```
